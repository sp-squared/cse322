Lecture 2: "HTTP"

GET method is invoked first when a webpage is first loaded. Host header is followed by the GET and is required!

  1. GET (WHAT YOU ARE GETTING)

    2. ex. GET/index.html HTTP/1.1

  2. HOST (WHERE ARE YOU GETTING)

    3. ex. Host: localhost:9999

    --> local host on port 9999

  3. Connection (ACTIVE?)
